# 🎩《资本家和梦想家》游戏说明书

---

## 🎯 1. 游戏简介

《资本家和梦想家》是一款基于经典电视节目《Deal or No Deal》改编的策略博弈游戏。  
👤 玩家扮演**梦想家**，从 26 个装有不同金额的箱子中选择一个，随后通过多轮开启其他箱子逐步缩小金额范围。  
💰 每一轮结束后，**资本家**会给出一个报价，试图以低价买走你手中的箱子。你的目标是在恰当的时机成交，尽可能获得更高的奖金，并赢得积分。  
📈 游戏还提供了**大富翁**和**大萧条**两种特殊模式，增加挑战与趣味。

## 📜 2. 游戏规则

### 2.1 🎁 初始设置
- 游戏共有 26 个箱子，编号 1～26，内部随机放置了以下金额之一（单位：元）：

```
0.1, 1, 5, 10, 25, 50, 75, 100, 200, 300, 400, 500, 750, 1000,
5000, 10000, 25000, 50000, 75000, 100000, 200000, 300000, 400000,
500000, 750000, 1000000
```

- 🎯 玩家首先选择一个箱子作为自己的“梦想箱子”（**不打开**，金额未知）。

### 2.2 📦 开箱流程
游戏分为若干轮，每轮必须按以下数量打开其他箱子：

| 🔢 轮数 | 📦 必须打开箱子数 |
|---------|------------------|
| 第1轮   | 6个 |
| 第2轮   | 5个 |
| 第3轮   | 4个 |
| 第4轮   | 3个 |
| 第5轮   | 2个 |
| 第6轮及以后 | 1个 |

- 🔓 打开的箱子金额会从**金额面板**上移除，不再可能出现在你的箱子中。
- 📊 你可以在任何时候看到当前剩余金额列表，以此推断自己箱子可能的价值。

### 2.3 🤝 资本家报价与决策
每轮开箱结束后，资本家会基于剩余金额面板计算一个报价（向下取整到百位），试图购买你的箱子。

**玩家此时有三个选项：**
- ✅ **Deal（成交）**：接受报价，游戏结束，你获得报价金额。
- ❌ **No Deal（拒绝）**：继续下一轮开箱。
- 💬 **Counter（还价）**：在全游戏过程中**只能使用一次**。你可以输入一个更高的价格向资本家还价。
  - ✔️ 如果资本家接受，则按你的还价成交，游戏结束。
  - 🚫 如果资本家拒绝，还价机会永久失效，你将继续开箱（也可选择接受原报价）。

> ⚠️ **注意**：当你选择 Deal 后，仍然可以使用还价机会；当你选择 No Deal 后，也可以直接还价。还价被拒后，可以继续开箱，也可以当场接受资本家原来的报价。

### 2.4 🏁 游戏结束
游戏在以下情况结束：
- 🎉 你接受了报价或还价成功。
- 📦 所有其他箱子都被打开，梦想箱子自动揭晓，你获得其中的实际金额。

## 🧮 3. 积分系统（总分 1000 分）

最终积分由两段组成，各占 500 分：

### 1️⃣ 第一段积分（0～500 分）
基于你**最终到手的奖金金额**（无论成交还是揭晓）。  
🏆 奖金越高，得分越接近 500 分；最低奖金 0.1 元对应 0 分。  
采用平方映射函数，使中高奖金段分数增长更平缓（例如 75000 元约得 350 分）。

### 2️⃣ 第二段积分（-500～500 分）
基于**交易盈亏** = 最终到手金额 − 梦想箱子实际金额。  
📈 **梦想家赚钱**（到手 > 实际）：快速增加正向分数，例如赚 5 万可得约 405 分。  
📉 **资本家赚钱**（到手 < 实际）：缓慢降低分数，例如亏 5 万约扣 77 分。  
⚠️ 极端亏损（如 100 元买到 100 万元箱子）会导致较大负分，体现风险与博弈。

📊 **总分** = 第一段积分 + 第二段积分，理想情况下最高 1000 分，极差交易可能出现负分。

## 🎮 4. 游戏模式

| 🎭 模式 | ✨ 修改内容 |
|--------|-------------|
| 🌟 **标准模式** | 使用基础金额分布，真实还原博弈乐趣。 |
| 💰 **大富翁模式** | 后 9 个箱子全部替换为 1,000,000 元，大幅提高获得巨款的概率。 |
| 📉 **大萧条模式** | 前 9 个箱子全部替换为 0.1 元，使箱子贬值，考验你的决策心理。 |

模式在游戏开始时选择。

## 🖱️ 5. 操作说明（图形界面版）

- 🚀 **启动游戏**：双击 `MillionGame.exe`（或运行 `main.py`）进入图形窗口。
- 🎯 **选择模式**：点击对应按钮。
- 🎁 **选择梦想箱子**：输入编号 1～26，点击“选定”。
- 📦 **开箱**：每轮按要求数量选择箱子编号并打开，界面会实时更新金额面板。
- 🤔 **决策**：资本家报价后，点击 ✅ **Deal** 接受报价、❌ **No Deal** 继续开箱，或 💬 **Counter** 输入还价金额。
- 🏁 **游戏结束**：显示详细结果和积分，可选择查看所有箱子实际金额，或重新开始。

## 📦 6. 打包为可执行文件

如果你想在未安装 Python 的 Windows 电脑上运行，可以自行打包：

1. 📥 安装 PyInstaller：`pip install pyinstaller`
2. 🛠️ 在项目目录执行：
   ```bash
   pyinstaller --onefile --noconsole --name MillionGame --icon=your_icon.ico main.py
   ```
   - `--onefile`：生成单个 exe 文件。
   - `--noconsole`：不显示命令行窗口（纯图形界面）。
   - `--icon`：设置程序图标（可选）。
3. 📁 打包好的程序在 `dist/` 文件夹中。

## 💡 8. 小提示

- 💎 还价机会只有一次，请在最有利的时机使用。
- 📊 当剩余金额面板中高额数字占多数时，你的箱子很可能是大金额，资本家的报价会较高；反之则报价较低。
- 🧠 可以通过分析剩余金额的分布，结合报价来判断是否应该成交。
- 💰 大富翁模式下资本家报价会偏保守，尽早成交可能更划算。
